# Настройка окружения

Порядок действий на своей машине, до первого запуска агента.

## 1. Базовые инструменты

- **Go** (последняя стабильная 1.2x) — https://go.dev/dl/
- **Docker + Docker Compose** — https://docs.docker.com/get-docker/
- **Git**
- **VS Code** + расширение "Go" (golang.go)

Проверка после установки:
```bash
go version
docker compose version
git --version
```

## 2. Claude Code

Установка (npm):
```bash
npm install -g @anthropic-ai/claude-code
```

Если версия отличается или команда не находится — свериться с актуальной документацией:
https://docs.claude.com

## 3. Структура репозитория

Создай пустой репозиторий и положи в его корень (в /) три файла, которые дальше пришлю:
- `CLAUDE.md` — обязательно в корне, Claude Code читает его автоматически при старте в этой папке.
- `PROJECT_BRIEF.md`
- `ROADMAP_SPRINTS.md`

```bash
mkdir notification-hub && cd notification-hub
git init
# сюда положить три .md файла
git add .
git commit -m "init: project docs"
```

## 4. Запуск

Открой папку проекта в VS Code, открой встроенный терминал (Ctrl+`) и запусти:
```bash
claude
```

Дальше внутри диалога с агентом просто скажи что-то вроде:
> Прочитай CLAUDE.md, PROJECT_BRIEF.md и ROADMAP_SPRINTS.md в корне репозитория. Начни со Спринта 1, Задача 1. После каждой задачи останавливайся и жди подтверждения перед следующей.

Никаких дополнительных "skills" (в смысле Anthropic Skills-фреймворка) для этого проекта не нужно — это обычный Go/Docker бэкенд, агент работает через свои штатные инструменты (bash, редактирование файлов). Skills нужны для другого класса задач (генерация docx/pptx/pdf и т.п.), тут неприменимо.

## 5. Важно про секреты

Firebase service account key и любые токены — никогда не коммить в git. Сразу создай `.gitignore` с `*.env`, `secrets/`, `*.json` для ключей (кроме package.json/go.mod, уточнить паттерн при добавлении).

## 6. Локалхост vs удалённый сервер

Спринты 1–3 полностью выполняются локально (docker-compose на твоей машине), деплой на VPS — только в Спринте 4. Явно скажи это агенту в первом сообщении, чтобы он не пытался сразу настраивать домены/TLS.
